I'll tell you all another trick with the glider.  While gliding, each glider has its own behavior when it comes to how it moves.  Most will start off with a nearly horizontal glide but will start to lose altitude gradually faster the longer you glide.  However, this glide curve can be reset using a quick maneuver mid-glide.  If you retract your glider and then pull it back out, it will reset the glide curve just like if you had pulled it out for the first time after jumping.  Using this technique, you can make it glide over a longer distance than you would with a regular glide. This trick only resets the curve, but it won't reset the stamina.
