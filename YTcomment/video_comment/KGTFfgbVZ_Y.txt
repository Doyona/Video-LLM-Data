It's NOT advisable to add nutrients (especially those with DAP in them) past the 9% ABV point, as you are in stage where the yeast will not use it, so it could end up staying in your mead and effecting the taste.  IF you add past the 9% point, then you would probably want to do so using Fermaid O, which is an organic product, that contains no DAP.  Fermaid O is primarily made up of yeast hulls and some vitamins, so will eventually settle out if not used.
