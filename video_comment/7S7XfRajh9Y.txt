Given the electrical changes in the security system etc. I would be concerned about the security system flooding the BUS of the car.  Those types of systems have the ability to not only send, but receive messages.  If the unit sends too much communication it can "flood" the system.  This is likely why resetting the ECU gets you by for a bit, but once the communication starts creating errors then the onboard system is unable to get proper communication.  In my mind, I would remove all 3rd party systems.  Check all grounds to ensure something wasn't missed and check all connectors to assure you don't have a poor connection.  Sometimes pins can move then the connectors are connected.  It's fussy work, but critical with electrical problems.
